ABOUT -- DOOM II MUSIC / GENERAL MIDI / MT-32 / ADLIB / OPL3

One MUS score, four original devices.

Doom II: Hell on Earth (id Software, 1994) stored its music as MUS lumps inside DOOM2.WAD and played them through the DMX sound library. SETUP.EXE let you choose your audio hardware, and the same Bobby Prince score was voiced differently for each. This jukebox plays four of those paths side by side: General MIDI (Roland SC-55), Roland MT-32 (via the MT2GM timbre set), AdLib (OPL2 FM) and Sound Blaster / OPL3 (FM). Each MUS lump is converted to a Standard MIDI File; use the Gen MIDI / MT-32 / AdLib / OPL3 filter to switch.

General MIDI: the Sound Canvas mix (premium path)

The General-MIDI path sends each MUS-converted score straight to a Roland SC-55 / SCC-1 Sound Canvas. It is fully multi-timbral — up to sixteen parts at once with standard GM instruments and a dedicated drum channel — so it is the cleanest, fullest version Doom offers. We render it through a cycle-accurate WebAssembly Nuked-SC55.

Roland MT-32: the MT2GM trick

The MT-32 isn't a General-MIDI device, so the Doom score normally sounds wrong on it. The fix is MT2GM — a SysEx set published with Doom's own composer, Bobby Prince. Before the music plays we upload MTGM.MID (64 GM-arranged user timbres + a new patch map and channel layout) and the standard GM drum kit (MTR-STND.MID), which rewrite the MT-32's sound memory into a General-MIDI arrangement. The unmodified GM score then plays on the re-voiced synth — the classic “MT-32 plays Doom” trick from the DOS days. Doom's MUS uses few enough channels that a single MT-32 can carry it.

We bake the MT2GM SysEx upload onto the front of every cue and play the result through the Munt CM-32L emulator shared with the other jukeboxes — so what you hear is a real MT-32 voiced for General MIDI, not a program remap.

AdLib & OPL3: the game's own FM voices

The two FM builds use Doom's own instruments. We parse GENMIDI — the DMX OP2 bank lump from DOOM2.WAD, 175 patches (128 melodic + percussion) — into a WOPL bank and render each score through a cycle-accurate Nuked-OPL3 (via libADLMIDI). The AdLib build uses each instrument's single 2-operator voice, exactly like a real OPL2 card. The Sound Blaster / OPL3 build is Doom's DMXOPTION=opl3 mode: the 32 GENMIDI instruments that carry a second voice play both at once, slightly detuned (a “pseudo 4-operator” pair), for the thicker, wider OPL3 sound.

How this works

The chain from the 1995 game data to your headphones:

DOOM2.WAD MUS lump-> Standard MIDI File-> GM score / MT2GM SysEx prelude / GENMIDI OP2 -> WOPL-> JavaScript timing-> Nuked-SC55 (GM) / Munt CM-32L (MT-32) / Nuked-OPL3 (AdLib + OPL3)-> synthesized sound-> modern headphones

The synthesis engine, ROM images and UI are shared with the sister TFX jukebox — only the MUS conversion, the GENMIDI->WOPL bank, the MT2GM prelude and this page are new.
